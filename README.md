# Movie Director Website

this repository/project is an website for movie director " Amir Taghzadeh " .

in this project we used React for front-end and Node Js for back-end ; for css framework we used bootstrap 5 and some custom css.
for a great director website we need some features like minimal and modern website, movies list, director info, some blog posts and some other features that makes website better than a sample website.
also website should be responsive and fast for all of devices like desktops (PC and Laptops), tablets and mobile devices (large, medium, large screen) .

in an director website, design is an important things that makes website more exciting for clients and users, so we will use Material Design for amazing design and SEO .
